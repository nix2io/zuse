Zuse Cheat Sheet


This page provides quick reference to all of the Zuse functions, group by their major functionality:

Compare

LTE If the left value is less than or equal to the right value
GT If the left value is greater than the right value
And If the left value and the right value is true
GTE If the left value is greater than or equal to the right value
Or If the left value or the right value is true
Eq If two values are equal
LT If the left value is less than the right value
Not No Description

Logic

Var Get the value of a variable
Let Define a variable
If If statement

Debug

Log Print text to stdout

Types

Argument The type for a function argument
Boolean Represent a boolean
Function Create an anonymous function
Text Represent text
Null Represent a Null value
Array Create an array of items
Number Represent a number
